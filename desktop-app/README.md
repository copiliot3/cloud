# CloudDrive Lumina — Desktop Server Manager

This is a premium, macOS/iOS-style frameless desktop application that lets you manage your background server and client processes dynamically. 

It communicates directly with your PM2 services using the programmatic backend API.

---

## Features
* **Real-Time Resource Monitor**: Consolidates CPU load, active system memory usage, and OS host properties.
* **Unified Process Control**: Restart, Stop, or Start both frontend and backend processes with a single click.
* **Direct Terminal Logs**: Select any running process to view the latest 50 lines of live log output.
* **Custom Accent Palette**: High-fidelity iOS glassmorphism cards that instantly synchronize with your chosen accent color theme.
* **Popup Dialog Modals**: Beautiful blurred overlays for service controls, avoiding accidental shutdowns.

---

## How to Run it

1. Make sure your backend server is running (port 3001).
2. Open a new terminal in this folder (`d:\web-projectttttttttttt\desktop-app`).
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

*(Requires Node.js and npm installed on the computer).*
