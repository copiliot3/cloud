## 🎮 How to Use Right Now

```
Step 1: Open Browser
        http://localhost:5173
        
Step 2: Navigate to Files
        Click "Local Disk (C:)" or (D:)
        
Step 3: Right-Click File
        Select "Share" from menu
        
Step 4: Choose Permission
        ⊙ Read-Only  (view only)
        ⊙ Read & Write (can modify)
        
Step 5: Copy Link
        Click copy button or Ctrl+C
        
Step 6: Share!
        Email, chat, or post the link
        
Step 7: Recipient Opens
        Opens link in browser
        Access based on permission
```
